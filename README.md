# Environmental Sensor Station Simulation and Analysis 
## (Domain: Cloud Computing)

## Set up virtual machine
 '''
 vagrant up  
 vagrant ssh  
 '''

 ## Migrate Database
 '''
 python manage.py migrate 
 '''

## In VM, run server
'''
 python manage.py runserver 0.0.0.0:8080  
 '''
## Home Page:
